Splay 的复杂度证明

记号

记 $siz(x)$ 表示 x 的子树大小，定义节点的势能函数 $\phi(x)=\log_2(siz(x))$，整棵 Splay 的势能 $\Phi=\sum\limits_{i=1}^n\phi(i)$。

splay

单旋

$$ \begin{aligned} \Delta\Phi&=\phi’(x)+\phi’(y)-\phi(x)-\phi(y)\ &=\phi’(y)-\phi(x)\ &<\phi’(x)-\phi(x) \end{aligned} $$

双旋

$$ \begin{aligned} \Delta\Phi&=\phi’(x)+\phi’(y)+\phi’(z)-\phi(x)-\phi(y)-\phi(z)\ &=\phi’(y)+\phi’(z)-\phi(x)- \phi(y)\ &<\phi’(x)+\phi’(z)-2\phi(x) \end{aligned} $$ 因为 $$ siz(x)+siz’(z)=2+siz(A)+siz(B)+siz(C)+siz(D)<siz’(x) $$ 所以 $$ siz(x)siz’(z)<\frac14siz’(x)^2 $$ 所以 $$ \phi(x)+\phi’(z)-2\phi’(x)=\log_2\left(\frac{siz(x)siz’(z)}{siz’(x)^2}\right)<\log_2(\frac14)=-2 $$ 即 $$ -(\phi(x)+\phi’(z)-2\phi’(x))-2>0 $$ 原式加上上式 $$ \Delta\Phi<\phi’(x)+\phi’(z)-2\phi(x)-(\phi(x)+\phi’(z)-2\phi’(x))-2=3(\phi’(x)-\phi(x))-2 $$ $$ \begin{aligned} \Delta\Phi&=\phi’(x)+\phi’(y)+\phi’(z)-\phi(x)-\phi(y)-\phi(z)\ &=\phi’(y)+\phi’(z)-\phi(x)-\phi(y)\ &<\phi’(y)+\phi’(z)-2\phi(x) \end{aligned} $$ 因为 $$ siz’(y)+siz’(z)<siz’(x) $$ 所以 $$ siz’(y)siz’(z)<\frac14siz’(x)^2 $$ 所以 $$ \phi’(y)+\phi’(z)-2\phi’(x)=\log_2\left(\frac{siz’(y)siz’(z)}{siz’(x)^2}\right)<\log_2\left(\frac14\right)=-2 $$ 即 $$ -(\phi’(y)+\phi’(z)-2\phi’(x))-2>0 $$ 原式加上上式 $$ \Delta\Phi<\phi’(y)+\phi’(z)-2\phi(x)-(\phi’(y)+\phi’(z)-2\phi’(x))-2=2(\phi’(x)-\phi(x))-2 $$ 双旋势能变化量中的 $-2$ 刚好和两次旋转相抵消， 相当于没有旋转消耗仅增加势能。

单旋的消耗没有办法被抵消，但仅在终点处进行最多 1 次。

设一次 splay 的起点为 x，终点为 y，中间的点可以全部消掉，所以势能增量为 $3(\phi(y)-\phi(x))<3\log_2n$。

因为势能永远 $\ge 0$，初始势能 $\le n\log_2 n$，所以 n 个点的 Splay 进行 m 次 splay 的旋转次数最多为 $(n+3m)\log_2 n+m$。

时间复杂度 $\mathcal O((n+m)\log n)$。

插入

找到前驱 splay 到根（$3\log_2 n$），把该点插入到前驱和右儿子之间（$\log_2(siz(x))$），再旋转一下（$\log_2n-\log_2(siz(x))$）。

旋转次数 $(n+4m)\log_2n+m$，时间复杂度 $O((n+m)\log n)$。

删除

找到该点 splay 到根（$3\log_2 n$），再找到前驱 splay 到根的右儿子（$3\log_2 n$），把该点右儿子挂到前驱的右儿子上（$\log_2(n-1)-\log_2(siz(x))$），删除该点（$-\log_2n$）。

旋转次数 $(n+6m)\log_2n+2m$，时间复杂度 $O((n+m)\log n)$。

Link Cut Tree 的复杂度证明

记号

$rsz(x)$：辅助树的子树大小，$rsz(x)=rsz(ls(x))+rsz(rs(x))+1$。

$vsz(x)$：辅助树子树内所有点的所有虚儿子的 $siz$ 之和，$vsz(ls(x))+vsz(rs(x))+\sum\limits_{y\in vs(x)}siz(y)$。

$siz(x)$：$rsz(x)+vsz(x)$。

access

可以发现 access 操作过程中在数棵 Splay 中 splay，类似这样：

定义势能函数 $\phi(x)=\log_2(siz(x))$，整个 LCT 的势能 $\Phi=\sum\limits_{i=1}^n\phi(i)$。

那么 splay 的势能增量之和即为 $3(\phi(b)-\phi(a))+3(\phi(d)-\phi(c))+3(\phi(f)-\phi(e))$，注意到我们定义的势能函数包括辅助树子树中所有点的 $siz$ 之和，所以 $\phi(a)\le\phi(b)<\phi(c)\le\phi(d)<\phi(e)\le\phi(f)$，那么全部消去，即得 $3(\phi(f)-\phi(a))<3\log_2n$。

势能永远 $\ge 0$，初始势能 $\le n\log_2 n$，所以 n 个点的 LCT 进行 m 次 access，双旋中的旋转次数最多为 $(n+3m)\log_2n$。

一次 access 还进行了最多为 splay 次数的单旋和虚实链切换。

如果 $siz(x)\ge siz(fa(x))$，那么我们称 x 为 $fa(x)$ 的重儿子，$fa(x)$ 的其他儿子为轻儿子。

定义势能函数 $\delta$ 表示实儿子和重儿子不同的节点个数。

考虑 access 过程中的一次 splay 后，当前 Splay 的根：

  • 它是轻儿子，$\delta$ 增量 $\le 1$；
  • 它是重儿子，$\delta$ 减少 1，虚实链切换的消耗被抵消。

因为一次 access 只会遇见最多 $\log_2n$ 个轻儿子，所以一次 access 的势能增加量最多为 $\log_2n$。

势能永远 $\ge 0$，初始势能 $\le n$，所以 n 个点的 LCT 进行 m 次 access 的单旋和虚实链切换的次数最多为 $n+m\log_2n$。

综上，n 个点的 LCT 进行 m 次 access 的旋转次数最多为 $(n+4m)\log_2n+n$，虚实链切换次数最多为 $n+m\log_2n$。

时间复杂度 $\mathcal O((n+m)\log_2n)$。

AAA Tree 的复杂度证明

记号

$ls(x)$：实 Splay 中的左儿子。

$rs(x)$：实 Splay 中的右儿子。

$vls(x)$：虚 Splay 中的左儿子。

$vrs(x)$：虚 Splay 中的右儿子。

$vs(x)$：虚儿子。

$rsz(x)$：实 Splay 中的子树大小，$rsz(x)=rsz(ls(x))+rsz(rs(x))+1$。

$vsz(x)$：虚 Splay 子树（不含）内所有点的 $siz$ 之和 + 虚儿子的 $siz$ + 实 Splay 子树（不含）中所有内的 $vsz$ 之和，$vsz(x)=siz(vls(x))+siz(vrs(x))+siz(vs(x))+siz(ls(x))+siz(rs(x))$。

$siz(x)$：$rsz(x)+vsz(x)$。

access

splay

可以发现 access 的过程中虚 Splay 和实 Splay 中的 splay 交替进行，类似这样：

定义节点的势能函数为 $\phi(x)=\log_2(siz(x))$。

那么 splay 的势能变化量之和即为 $3(\phi(b)-\phi(a))+3(\phi(c)-\phi(b))+3(\phi(e)-\phi(d))+3(\phi(f)-\phi(e))+3(\phi(h)-\phi(g))+3(\phi(i)-\phi(h))$，注意到我们定义的势能函数包括实 Splay 子树中所有点的 $siz$ 之和，所以 $\phi(a)\le \phi(b)\le \phi(c)<\phi(d)\le \phi(e)\le \phi(f)<\phi(g)\le \phi(h)\le \phi(i)$，那么全部消去，即得 $3(\phi(i)-\phi(a))<3\log_2n$。

势能永远 $\ge 0$，初始势能 $\le n\log_2n$，所以 n 个点的 AAA Tree 进行 m 次 access，虚 Splay 中 splay 的旋转次数最多为 $(n+3m)\log_2n$，时间复杂度 $\mathcal O((n+m)\log n)$。

虚 Splay 中插入 & 删除

将 access 中换右儿子的操作分类处理。

  • 要换上的右儿子为空：仅将当前的右儿子插入虚 Splay 中，一次 access 仅有一次插入操作；
  • 当前右儿子为空：仅将要换上的右儿子从虚 Splay 中删除；
  • 当前右儿子和要换上的右儿子都非空：交换两个节点。

右儿子为空即为链尾，且操作后链尾数量减少 1，而链尾数量仅可通过 link 操作每次增加 1，所以删除次数不超过 link 次数。

所以 n 个点的 AAA Tree 进行 m 次 access 的插入、删除次数最多为 n，所含的旋转次数最多为 $12n\log_2n+3n$，时间复杂度 $\mathcal O(n\log n)$。

综上，旋转次数 $(13n+3m)\log_2n+3n$，时间复杂度 $\mathcal O((n+m)\log n)$。

一种五叉 AAA Tree 的实现：

// 5 叉 AAA Tree
// 常数和 3 叉 AAA Tree 相近但不需要 2 倍空间，代码长度相近 
#include<bits/stdc++.h>
using namespace std;

typedef long long ll;
typedef unsigned long long ull;
typedef long double ldb;

#define gc getchar

int gi() { // cannot input -2147483648
	int x = 0, c = gc();
	bool f = 0;
	for(; !isdigit(c); c = gc()) if(c == '-') f = 1;
	for(; isdigit(c); c = gc()) x = x * 10 + (c & 15);
	return f ? -x : x;
}

const int N = 100005;
const int INF = (1u << 31) - 1;

struct Tag {
	int mul, add;
	Tag(): mul(1), add(0) {}
	Tag(int mul, int add): mul(mul), add(add) {}
	bool exis() { return (mul != 1 || add != 0); } 
};
int operator + (const int &l, const Tag &r) {
	return l * r.mul + r.add;
}
int& operator += (int &l, const Tag &r) {
	return l = l + r;
}
Tag operator + (const Tag &l, const Tag &r) {
	return Tag(l.mul * r.mul, l.add * r.mul + r.add);
}
Tag& operator += (Tag &l, const Tag &r) {
	return l = l + r;
}
struct Data {
	int siz, sum, mn, mx;
	Data(): siz(0), sum(0), mn(INF), mx(-INF - 1) {}
	Data(int x): siz(1), sum(x), mn(x), mx(x) {}
	Data(int siz, int sum, int mn, int mx): siz(siz), sum(sum), mn(mn), mx(mx) {}
};
Data operator + (const Data &l, const Data &r) {
	return Data(l.siz + r.siz, l.sum + r.sum, min(l.mn, r.mn), max(l.mx, r.mx));
}
Data& operator += (Data &l, const Data &r) {
	return l = l + r;
}
Data operator + (const Data &l, const Tag &r) {
	return Data(l.siz, l.sum * r.mul + l.siz * r.add, l.mn + r, l.mx + r);
}
Data operator += (Data &l, const Tag &r) {
	if(!l.siz) return l;
	return l = l + r;
}

int fa[N], ch[N][5], val[N], rev[N];
Data datr[N], datv[N];
Tag tagr[N], tagv[N];

#define ls(x) ch[x][0]
#define rs(x) ch[x][1]
#define vls(x) ch[x][2]
#define vrs(x) ch[x][3]
#define vs(x) ch[x][4]
#define sum(x) (datr[x] + datv[x])

void pushup(int x) { // assume that x != 0
	datr[x] = datr[ls(x)] + datr[rs(x)] + Data(val[x]);
	datv[x] = datv[ls(x)] + datv[rs(x)] + sum(vls(x)) + sum(vrs(x)) + sum(vs(x));
}
void flip(int x) {
	if(!x) return;
	rev[x] ^= 1;
	swap(ls(x), rs(x));
}
void push_real(int x, Tag v) {
	if(!x) return;
	val[x] += v;
	datr[x] += v;
	tagr[x] += v;
}
void push_virt(int x, Tag v) {
	if(!x) return;
	datv[x] += v;
	tagv[x] += v;
}
void push_all(int x, Tag v) {
	if(!x) return;
	push_real(x, v);
	push_virt(x, v);
}
void pushdown(int x) {
	if(rev[x]) {
		flip(ls(x));
		flip(rs(x));
		rev[x] = 0;
	}
	if(tagr[x].exis()) {
		push_real(ls(x), tagr[x]);
		push_real(rs(x), tagr[x]);
		tagr[x] = Tag();
	}
	if(tagv[x].exis()) {
		push_virt(ls(x), tagv[x]);
		push_virt(rs(x), tagv[x]);
		push_all(vls(x), tagv[x]);
		push_all(vrs(x), tagv[x]);
		push_all(vs(x), tagv[x]);
		tagv[x] = Tag();
	}
}
bool notroot(int x, int k) {
	return (ch[fa[x]][k] == x || ch[fa[x]][k + 1] == x);
}
int chk(int x) {
	for(int i = 0; i <= 3; ++i) {
		if(ch[fa[x]][i] == x) return i;
	}
	return 4;
}
int chk(int x, int k) { // assume that fa[x] != 0
	return k + (ch[fa[x]][k + 1] == x);
}
void connect(int f, int k, int x) {
	fa[x] = f; ch[f][k] = x;
}
void rotate(int x, int k) { // assume that x is not root
	int y = fa[x], z = fa[y], w = chk(x, k);
	connect(y, w, ch[x][w ^ 1]);
	if(k == 0 && !notroot(y, 0)) {
		connect(x, 2, vls(y));
		connect(x, 3, vrs(y));
		vls(y) = vrs(y) = 0;
	}
	connect(z, chk(y), x);
	connect(x, w ^ 1, y);
	pushup(y);
}
void pushpath(int x, int k) {
	if(notroot(x, k)) {
		pushpath(fa[x], k);
	}
	pushdown(x);
}
void splay(int x, int k) { // 0 to real, 2 to virtual
	pushpath(x, k);
	while(notroot(x, k)) {
		int y = fa[x];
		if(notroot(y, k)) rotate(chk(x, k) == chk(y, k) ? y : x, k);
		rotate(x, k);
	}
	pushup(x);
}
int getfa(int x) {
	splay(x, 2);
	return fa[x];
}
void del(int x) { // assume that x is the root of the virtual Splay 
	if(!x || !fa[x]) return;
	pushdown(x);
	if(!vls(x) || !vrs(x)) { 
		connect(fa[x], 4, vls(x) | vrs(x));
		pushup(fa[x]);
		fa[x] = vls(x) = vrs(x) = 0;
		pushup(x);
		return;
	}
	int lef = vls(x);
	while(vrs(lef)) {
		lef = vrs(lef);
	}
	splay(lef, 2);
	connect(lef, 3, vrs(x));
	pushup(lef);
	fa[x] = vls(x) = vrs(x) = 0;
	pushup(x);
}
void ins(int x, int y) { // x -> y
	if(!x || !y) return;
	pushdown(y);
	if(!ch[y][4]) {
		connect(y, 4, x);
		pushup(y);
		return;
	}
	y = ch[y][4];
	while(vls(y)) {
		pushdown(y);
		y = vls(y);
	}
	pushdown(y);
	connect(y, 2, x);
	splay(x, 2);
}
int access(int x) {
	int y = 0;
	while(x) {
		splay(x, 0);
		if(y && rs(x)) { // 交换两个点，单 log 的关键
			pushdown(rs(x));
			pushdown(y);
			connect(rs(x), 2, vls(y));
			connect(rs(x), 3, vrs(y));
			vls(y) = vrs(y) = 0;
			pushup(rs(x));
			pushup(y);
			connect(x, 4, rs(x));
		}
		else {
			del(y);
			ins(rs(x), x);
		}
		connect(x, 1, y);
		pushup(x);
		y = x;
		x = getfa(x);
	}
	return y;
}
int lca(int x, int y) {
	access(x);
	return access(y);
}
void makeroot(int x) {
	access(x);
	splay(x, 0);
	flip(x);
}
int findroot(int x) {
	access(x);
	splay(x, 0);
	while(ls(x)) {
		x = ls(x); // LCT 保证 rev 标记不会出现在 Splay 根的右子树里 
	}
	splay(x, 0);
	return x;
}
void split(int x, int y) {
	makeroot(x);
	access(y);
	splay(y, 0);
}
void link(int x, int y) {
	makeroot(x);
	access(y);
	splay(y, 0);
	ins(x, y);
}
void cut(int x) {
	access(x);
	splay(x, 0);
	fa[ls(x)] = 0;
	ls(x) = 0;
	pushup(x);
}
void modifychain(int x, int y, Tag v) {
	split(x, y);
	push_real(y, v);
}
Data querychain(int x, int y) {
	split(x, y);
	return datr[y];
}
void modifysubtree(int x, Tag v) {
	access(x);
	splay(x, 0);
	val[x] += v;
	push_all(vs(x), v);
	pushup(x);
}
Data querysubtree(int x) {
	access(x);
	splay(x, 0);
	return Data(val[x]) + sum(vs(x));
}

int e[N][2];

int main() {
	int n = gi(), m = gi();
	for(int i = 1; i < n; ++i) {
		e[i][0] = gi();
		e[i][1] = gi();
	}
	for(int i = 1; i <= n; ++i) {
		val[i] = gi();
		pushup(i);
	}
	for(int i = 1; i < n; ++i) {
		link(e[i][0], e[i][1]);
	}
	int r = gi();
	while(m--) {
		int op = gi();
		if(op == 0) {
			int x = gi(), y = gi();
			makeroot(r);
			modifysubtree(x, Tag(0, y));
		}
		else if(op == 1) {
			r = gi();
		}
		else if(op == 2) {
			int x = gi(), y = gi(), z = gi();
			modifychain(x, y, Tag(0, z));
		}
		else if(op == 3) {
			int x = gi();
			makeroot(r);
			printf("%d\n", querysubtree(x).mn);
		}
		else if(op == 4) {
			int x = gi();
			makeroot(r);
			printf("%d\n", querysubtree(x).mx);
		}
		else if(op == 5) {
			int x = gi(), y = gi();
			makeroot(r);
			modifysubtree(x, Tag(1, y));
		}
		else if(op == 6) {
			int x = gi(), y = gi(), z = gi();
			modifychain(x, y, Tag(1, z));
		}
		else if(op == 7) {
			int x = gi(), y = gi();
			printf("%d\n", querychain(x, y).mn);
		}
		else if(op == 8) {
			int x = gi(), y = gi();
			printf("%d\n", querychain(x, y).mx);
		}
		else if(op == 9) {
			int x = gi(), y = gi();
			makeroot(r);
			if(lca(x, y) == x) continue;
			cut(x);
			makeroot(y);
			ins(x, y);
		}
		else if(op == 10) {
			int x = gi(), y = gi();
			printf("%d\n", querychain(x, y).sum);
		}
		else {
			int x = gi();
			makeroot(r);
			printf("%d\n", querysubtree(x).sum);
		}
	}
	return 0;
}
