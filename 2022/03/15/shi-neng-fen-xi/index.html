Splay 的复杂度证明

记号

记 \(siz(x)\) 表示 x 的子树大小，定义节点的势能函数 \(\phi(x)=\log_2(siz(x))\)，整棵 Splay 的势能 \(\Phi=\sum\limits_{i=1}^n\phi(i)\)。

splay

单旋

\[ \begin{aligned} \Delta\Phi&=\phi'(x)+\phi'(y)-\phi(x)-\phi(y)\\ &=\phi'(y)-\phi(x)\\ &<\phi'(x)-\phi(x) \end{aligned} \]

双旋

\[ \begin{aligned} \Delta\Phi&=\phi'(x)+\phi'(y)+\phi'(z)-\phi(x)-\phi(y)-\phi(z)\\ &=\phi'(y)+\phi'(z)-\phi(x)- \phi(y)\\ &<\phi'(x)+\phi'(z)-2\phi(x) \end{aligned} \]

因为

\[ siz(x)+siz'(z)=2+siz(A)+siz(B)+siz(C)+siz(D)<siz'(x) \]

根据基本不等式有

\[ siz(x)+siz'(z)\le\frac{\left(siz(x)+siz'(z)\right)^2}{4} \]

所以

\[ siz(x)siz'(z)<\frac14siz'(x)^2 \]

所以

\[ \phi(x)+\phi'(z)-2\phi'(x)=\log_2\left(\frac{siz(x)siz'(z)}{siz'(x)^2}\right)<\log_2(\frac14)=-2 \]

即

\[ -(\phi(x)+\phi'(z)-2\phi'(x))-2>0 \]

原式加上上式

\[ \Delta\Phi<3(\phi'(x)-\phi(x))-2 \]

\[ \begin{aligned} \Delta\Phi&=\phi'(x)+\phi'(y)+\phi'(z)-\phi(x)-\phi(y)-\phi(z)\\ &=\phi'(y)+\phi'(z)-\phi(x)-\phi(y)\\ &<\phi'(y)+\phi'(z)-2\phi'(x) \end{aligned} \]

因为

\[ siz'(y)+siz'(z)<siz'(x) \]

所以

\[ siz'(y)siz'(z)<\frac14siz'(x)^2 \]

所以

\[ \phi'(y)+\phi'(z)-2\phi'(x)=\log_2\left(\frac{siz'(y)siz'(z)}{siz'(x)^2}\right)<\log_2\left(\frac14\right)=-2 \]

即

\[ -(\phi'(y)+\phi'(z)-2\phi'(x))-2>0 \]

原式加上上式

\[ \Delta\Phi<2(\phi'(x)-\phi(x))-2 \]

综上双旋一次势能减少 2，因为势能不会小于 0，所以双旋中的旋转次数 \(\le\) 总势能。

单旋的消耗没有办法被抵消，但仅在终点处进行最多 1 次。

设一次 splay 的起点为 x，终点为 y，中间的点可以全部消掉，所以势能增量 \(\Delta\Phi\le3(\phi(y)-\phi(x))\le3\log_2n\)。

初始势能 \(\Phi_0\le n\log_2n\)，所以 n 个点的 Splay 进行 m 次 splay 的旋转次数最多为 \((n+3m)\log_2 n+m\)。

时间复杂度 \(\mathcal O((n+m)\log n)\)。

插入

找到前驱 splay 到根（\(3\log_2 n\)），把该点插入到前驱和右儿子之间（\(\log_2(siz(x))\)），再旋转一下（\(\log_2n-\log_2(siz(x))\)），一次插入的势能增量 \(\Delta\Phi\le4\log_2n\)

旋转次数上界 \((n+4m)\log_2n+2m\)，时间复杂度 \(O((n+m)\log n)\)。

删除

找到该点 splay 到根（\(3\log_2n\)），再找到前驱 splay 到根的右儿子（\(3\log_2(siz(ls(x)))\)），把该点右儿子挂到前驱的右儿子上（\(\log_2(n-1)-\log_2(siz(ls(x)))\)），删除该点（\(-\log_2n\)），一次删除的势能增量 \(\Delta\Phi\le5\log_2n\)

旋转次数上界 \((n+5m)\log_2n+2m\)，时间复杂度 \(O((n+m)\log n)\)。

Link Cut Tree 的复杂度证明

记号

\(fa(x)\)：x 在辅助树上的父亲，如果 x 为辅助树的根，则为 x 的虚父亲。

\(siz(x)\)：所有能通过不断跳到 \(fa(x)\) 来到达 x 的点的数量。

只要 \(fa(x)\) 无环，则 \(siz(x)\le n\) 对任意 x 恒成立，且任意 \(fa\) 链上的 \(siz\) 从下到上单调递增。

access

可以发现 access 操作在数棵 Splay 中 splay，类似这样：

定义节点的势能函数 \(\phi(x)=\log_2(siz(x))\)，整个 LCT 的势能 \(\Phi=\sum\limits_{i=1}^n\phi(i)\)。

那么 splay 的势能增量之和 \(\Delta\Phi\le3(\phi(b)-\phi(a))+3(\phi(d)-\phi(c))+3(\phi(f)-\phi(e))\)，根据势能函数的定义有 \(\phi(a)\le\phi(b)<\phi(c)\le\phi(d)<\phi(e)\le\phi(f)\)，放缩消去中间量得 \(\Delta\Phi<3(\phi(f)-\phi(a))\le3\log_2n\)。

初始势能 \(\Phi_0\le n\log_2n\)，所以 n 个点的 LCT 进行 m 次 access，双旋中的旋转次数最多为 \((n+3m)\log_2n\)。

一次 access 还进行了最多为 splay 次数的单旋和虚实链切换。

如果 \(siz(x)\ge \frac{siz(fa(x)}{2})\)，那么我们称 x 为 \(fa(x)\) 的重儿子，\(fa(x)\) 的其他儿子为轻儿子。

定义势能函数 \(\delta\) 表示实儿子和重儿子不同的节点个数。

考虑 access 过程中的一次 splay 后，当前 Splay 的根：

  • 它是轻儿子，\(\delta\) 增量 \(\le 1\)；
  • 它是重儿子，\(\delta\) 减少 1，虚实链切换的消耗被抵消。

因为一次 access 只会遇见最多 \(\log_2n\) 个轻儿子，所以一次 access 的势能增加量最多为 \(\log_2n\)。

初始势能 \(\Phi_0\le n\)，所以 n 个点的 LCT 进行 m 次 access 的单旋和虚实链切换的次数最多为 \(m\log_2n+n\)。

综上，旋转次数上界 \((n+4m)\log_2n+n\)，虚实链切换次数上界 \(m\log_2n+n\)，时间复杂度 \(\mathcal O((n+m)\log_2n)\)。

Splay 维护 LCT 虚儿子的复杂度证明

记号

\(fa(x)\)：x 在实 Splay 上的父亲，如果 x 为其所在实 Splay 的根，则为 x 在虚 Splay 上的父亲（虚 Splay 根的父亲即为该虚 Splay 集体的虚父亲）。

\(siz(x)\)：所有能通过不断跳到 \(fa(x)\) 来到达 x 的点的数量。

所以 n 个点的 LCT 进行 m 次 access 的旋转次数最多为 \((n+5m)\log_2n+2n\)。

虚 Splay 中插入 & 删除

将 access 中换右儿子的操作分类处理。

  • 要换上的右儿子为空：仅将当前的右儿子插入虚 Splay 中，一次 access 仅有一次插入操作；
  • 当前右儿子为空：仅将要换上的右儿子从虚 Splay 中删除；
  • 当前右儿子和要换上的右儿子都非空：交换两个节点的位置。

右儿子为空即为链尾，且操作后链尾数量减少 1，而链尾数量仅可通过 link 操作每次增加 1，所以删除次数不超过 link 次数。

所以 n 个点的 AAA Tree 进行 m 次 access 的插入、删除次数最多为 m，所含的旋转次数最多为 \((2n+9m)\log_2n+4m\)。

综上，旋转次数上界 \((3n+14m)\log_2n+2n+4m\)，时间复杂度 \(\mathcal O((n+m)\log n)\)。
